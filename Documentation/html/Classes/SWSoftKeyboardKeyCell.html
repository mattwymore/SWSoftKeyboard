SWSoftKeyboardKeyCell Class Reference

Inherits from NSActionCell
Declared in SWSoftKeyboardKeyCell.h

Overview

Instances of SWSoftKeyboardKey represent individual keyboard keys. They contain certain state-related variables (isSticky, stateLabels, stateValues) as well as positioning values (frame).

stateLabels

The labels this key uses for certain keyboard/sticky states.

@property (nonatomic, strong) NSDictionary *stateLabels

Discussion

State dictionaries are stored such that the first dictionary key is the keyboard key’s sticky state (whether or not the key is stuck), and the second dictionary key is the keyboard’s layout state.
